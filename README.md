# OOP Learning App

Welcome to the OOP Learning App. This application is designed to help users learn Object-Oriented Programming concepts through an interactive interface.

## Prerequisites

- Node.js
- npm or yarn
- Expo Go (if testing on mobile)

## Setup Instructions

1. Clone the repository to your local machine.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add any necessary environment variables (such as `EXPO_PUBLIC_GEMINI_API_KEY`). **Note:** The `.env` file is excluded from version control to protect your private keys.
4. Start the application:
   ```bash
   npx expo start
   ```

## Key Technologies

- React Native (Expo)
- TypeScript
- Gemini API (AI integration)
